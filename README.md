# Akselerja

Platform AI job matching, skill assessment, dan upskilling untuk tenaga kerja Indonesia.

## Tech stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- General Sans via Fontshare CDN

## Development

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
